# database-repository

This repository contains SQL scripts, schema designs, and examples created for practicing and mastering database concepts using MySQL and Oracle Database.

The goal of this project is to build strong database skills required for real-world backend development and interviews.

Topics Covered

• Table creation and schema design
• CRUD operations
• Joins and subqueries
• Functions
• Stored procedures
• Packages (Oracle)
• Views and Materialized Views
• Indexing and performance optimization
• ETL examples
• Database links (Oracle ↔ MySQL)

Technologies

MySQL  
Oracle Database  
SQL / PL-SQL
